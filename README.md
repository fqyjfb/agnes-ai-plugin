# Agnes AI - ToolBox 插件

集成 Agnes AI 服务的智能助手插件，提供 AI 聊天、字体生成、图片生成、视频生成等功能。

## 功能特性

- **AI 助手聊天**：支持与 Agnes AI 进行智能对话，流式响应实时显示
- **字体生成器**：84 种艺术字体风格，支持自定义文字、尺寸、背景颜色、负向提示词等参数
- **图片生成**：通过 AI 生成高质量图片，支持多种尺寸和模型
- **视频生成**：支持 AI 视频生成任务提交和状态跟踪
- **历史记录**：管理生成的图片、视频、字体历史，支持下载、复制、删除操作
- **角色预设**：自定义 AI 角色设定，支持创建、编辑、删除角色预设
- **设置管理**：配置 API Key 和服务地址，支持密钥显示/隐藏
- **多对话管理**：支持创建多个对话，切换对话上下文
- **主题适配**：自动跟随 ToolBox 应用的浅色/深色主题

## 使用方法

### AI 助手聊天

1. **新建对话**：点击侧边栏的"新对话"按钮
2. **选择角色**：从角色预设列表中选择一个角色（可选）
3. **发送消息**：在输入框中输入内容，按 Enter 键或点击发送按钮
4. **重新生成**：鼠标悬停在助手回复上，点击重新生成按钮
5. **复制消息**：点击复制按钮复制消息内容

### 字体生成

1. **选择分类**：从左侧选择字体分类（时尚体、极简体、手写体等）
2. **选择样式**：点击具体的字体样式缩略图
3. **输入文字**：在输入框中输入想要生成的文字（2-10字）
4. **设置参数**：
   - 选择输出尺寸：512px ~ 1536px
   - 选择背景颜色：白色、黑色、透明、渐变
   - 输入种子值（可选）：用于控制生成结果的一致性
   - 输入负向提示词（可选）：排除不想要的效果
5. **生成字体**：点击"生成字体"按钮
6. **管理结果**：支持预览、下载、复制、删除操作

### 历史记录

1. **切换标签**：选择图片、视频、字体标签查看对应历史
2. **预览**：点击缩略图查看大图
3. **下载**：点击下载按钮保存到本地
4. **复制**：点击复制按钮复制图片链接
5. **删除**：点击删除按钮移除记录

### 角色预设

1. **新建角色**：点击"新建角色"按钮
2. **填写信息**：
   - 角色名称
   - 角色图标（emoji）
   - 角色描述
   - 系统提示词（定义 AI 的角色和行为）
3. **启用角色**：点击勾选按钮启用角色
4. **编辑角色**：点击编辑按钮修改角色信息
5. **删除角色**：点击删除按钮移除角色（默认角色不可删除）

### 设置

1. **配置 API Key**：输入 Agnes AI API Key
2. **配置服务地址**：默认 `https://apihub.agnes-ai.com`
3. **保存设置**：点击"保存"按钮
4. **重置设置**：点击"重置"按钮恢复默认值

## 快捷键

- **发送消息**：Enter 键
- **复制消息**：点击复制按钮
- **重新生成**：鼠标悬停在助手回复上点击重新生成按钮

## 开发

```bash
# 安装依赖
npm install

# 构建插件
npm run build

# 构建产物位于 dist/index.js
```
